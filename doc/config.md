# config

此页面介绍config设置的各个项目。

|项目|内容|
|----|----|
|`version`|版本号|
|`teamfile`|自动战斗使用的`team.json`文件，可在`config`目录下新建新的`teamjson`文件并设置。|
|`domain_times`|刷秘境的次数|
|`isLiYueDomain`|挑战部分石化古树被墙壁阻挡视野的秘境(大多位于璃月)时，设置为`true`，否则为`false`|
|`device`|yolox与paddleocr运算使用的设备，如果安装了cudnn则设为`'gpu'`，否则设为`'cpu'`|
|`debug`|是否启用debug模式|
