# genshin_impact_assistant 原神助手  

## 使用方法

1. 下载`tags`里最新版本的`genshin_assistant.zip`并解压
2. 下载`tags`里的`environment_cpu`或`environment_gpu`,并全部复制到解压后的`genshin_assistant`目录中
3. 启动原神后，以管理员权限运行`genshin_assistant.exe`
